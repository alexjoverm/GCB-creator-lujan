  • Linea 466: Sustituir por "var isCorrect = (correctAnswerRegex == textValue);"

Ficheros

GCB creator implementa varios algoritmos para la comprobación de los ficheros que se suben y/o abren. Si estos ficheros son creados desde la aplicación, no hay problema. Pero sino, la aplicación debería informarte de errores de sintaxis, o de nombre de ficheros.

Aún así, por cuestiones de seguridad, sigue estos consejos:

  • No nombres los ficheros con símbolos o acentos (sí que se permiten - ó _).
  • Revisa bien los ficheros de actividades o exámenes si los has creado a mano. Puede que la aplicación no detecte algún error que no se ha comprobado hasta ahora.

Toolbar

El toolbar es la barra de herramientas superior que podemos encontrar en las páginas de Actividad y Examen

Hay dos grupos de botones:

  • Generales: En este orden, la funcionalidad es Borrar preguntas, Descargar fichero, Subir fichero, Colapsar preguntas.
  • Creadores: Sirven para crear algún tipo de pregunta. Estos botones también se pueden encontrar pulsando en el espacio entre preguntas

Ordenar

Se pueden ordenar las preguntas, arrastrandolas y soltandolas en el hueco que se quiera. Para ello, hay que pulsar sobre la parte izquierda del recuadro de la pregunta.

Para ordenar en algunos dispositivos táctiles, como un iPad, se debe hacer click primero para seleccionar la pregunta. Entonces ya se podrá arrastrar.

Para mejorar la usabilidad de la aplicación, se permite insertar las preguntas en una posicion concreta, pulsando sobre el símbolo de situado entre los bloques de preguntas.

Editar

Hay campos que requieren de mucho texto, para los que se proveen un botón de editar, que abre un editor más grande donde se puede editar de forma más cómoda. Además de que se puede ampliar la altura de esos campos hasta cierto límite, ya que algunos no requerirán tanto texto.

Además, hay campos que pueden ser de texto con formato. Estos se distinguen fácilmente, ya que cuando se abre el editor completo, aparecerá una barra de herramientas de formato

Para los de texto sin formato, no aparecerá la barra de herramientas

Auto-guardado

La aplicación guarda automáticamente cualquier cambio que hagas. De hecho, cuando habras la página para editar, te aparecerá un mensaje que te informará de que hay una versión guardada, por si quieres continuarla

Titulo

Es importante poner un título correcto, ya que sino no podremos descargar el fichero. Para ello, prestar atención al formato que indica el campo cuando está vacío. Además aparecerán mensajes de validación una vez pulsemos el botón de descargar.

Aquí se puede comprobar la diferencia entre un título incorrecto y uno correcto

Actividad

En una actividad, puedes elegir entre 3 tipos de preguntas, además de poder incluir texto con formato (HTML) entre preguntas. Todas las preguntas tienen un campo de Pregunta, por lo que se omitirá su mención posteriormente.

HTML

Este no es una pregunta, y por lo tanto no se evalúa como tal. Simplemente es texto con formato que puede ir entre preguntas

Texto libre

Pregunta de texto libre. Los campos que la forman son:

  • Respuesta: es dada en formato de texto, no sensible a mayúsculas/minúsculas.
  • Feedback correcto: texto que se muestra cuando se acierta la pregunta
  • Feedback incorrecto: texto que se muestra cuando no se acierta

Multiple elección

Pregunta donde se pueden añadir un máximo de 6 respuestas seleccionables. Más de 1 puede ser correcta.

Los campos que la forman son:

  • Respuesta: texto de la respuesta.
  • Feedback: para esa respuesta específica.
  • Se mostrarán botones para marcar seleccionada, quitar y añadir respuesta.

Grupo multiple

Este caso es un conjunto de preguntas de las anteriores mencionadas, con alguna variación. En este caso el feedback es para todas las preguntas en general, y se pueden seleccionar cuántas preguntas debe contestar bien como mínimo, para que se considere correcta.

Los campos que la forman son:

  • Feedback correcto: se muestra cuando se superan el mínimo número de respuestas correctas.
  • Feedback incorrecto: se muestra cuando no se superan el mínimo número de respuestas correctas.
  • Lista de preguntas: cada pregunta tiene sus propios campos. Además tiene multi-seleccionable, que permite que se pueda seleccionar más de una respuesta correcta.

Examen

El modelo de examen es muy parecido al de actividad, con ligeros contrastes. Los tipos de preguntas son 3, y son similares.

Un examen tiene los siguientes campos:

  • Preámbulo: es como el enunciado que introduce el examen.
  • Autocorregir: si se marca, permitirá al alumno ver si las respuestas son correctas.
  • Lista de preguntas: tienen el campo de pregunta, y la de unidad-lección a la que pertenece. El formato de este último debe ser "número.número", tal como 9.3. Además de estos dos campos, cada pregunta tiene el suyo propio de respuesta.

Numérica

La respuesta debe ser numérica, permitiendo decimales separados por un punto.

Texto libre

Respuesta textual, sin diferenciar entre mayúsculas y minúsculas.

Multiple elección

En este caso se pueden añadir varias respuestas, de la que se seleccionará una correcta.
